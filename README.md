# dev-env-jenkins

Jenkins in Docker.

## Requirements

Docker.

## Getting Started

```bash
yarn create-network
yarn start
```

Open in browser: http://localhost:8080

To stop:

```bash
yarn stop
```

To uninstall (remove docker volumes + network):

```bash
yarn uninstall
```

Other commands

```bash
yarn ps
yarn logs
yarn logs -f
```
